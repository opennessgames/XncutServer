# 新手体验Java 30分钟总结：
1. 缜密/细节
    > new/close时很多情况下要try-catch。

2. 第二：苛刻/毅力
    > 错误/警告多，try-catch可能会满天飞。

3. 生命线/信誉
    > Java毕竟是企业级/服务器语言，7*24运转, 前两条就是为了守住生命线。

4. 没有头文件这个概念
    > 类与实现一体

5. GC/自动
    > 没有C/C++的手动管理内存，由GC管理
